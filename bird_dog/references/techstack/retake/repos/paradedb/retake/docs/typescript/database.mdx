---
title: Database
---

The `Database` class is used to store Postgres database credentials.

## Instantiation

```typescript Typescript
import { Database } from "retake-search"

const database = new Database({
  host: "***",
  user: "***",
  password: "***",
  dbName: "***",
  port: 5432,
})
```

  Postgres password
